== Description ==

This plugin adds Geesee chat to your WordPress blog. <a href="http://www.geesee.com/?source=wordpress_plugin_repository" target="_blank">Geesee</a> chat is free cross-site chat for any website or blog. You can check out Geesee in action <a href="http://www.geesee.com/?source=wordpress_plugin_repository" target="_blank">here</a>.

**Features**:

* Add chat to post or page
* Change chat width, height
* Change chat default chat room

== Installation ==

1. Upload `wp_geesee_chat` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Options/Geesee Chat' for settings and usage instructions
